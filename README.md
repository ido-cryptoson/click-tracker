# Click Tracker

A free link click tracker built with Next.js, Vercel, and Supabase.

## Features

- Create trackable short links
- Track clicks with detailed analytics:
  - Country
  - Browser
  - Device type (mobile/desktop/tablet)
  - Referrer
  - Timestamp
- Simple dashboard to view stats
- 100% free (using Vercel and Supabase free tiers)
